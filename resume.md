Neighbor.ly | React, TailwindCSS, Node.js, Express.js, PostgreSQL, Prisma, JWT, Git
• Developed a full-stack web application for comprehensive society management, facilitating seamless interaction between residents and administration.
• Architected robust RESTful APIs using Node.js with Express.js and managed data persistently with PostgreSQL and Prisma ORM.
• Implemented secure authentication and authorization using JWT tokens and bcrypt for distinct user (resident) and admin roles.
• Designed and built an intuitive user interface with React, featuring dynamic routing and state management for optimal user experience.
• Styled a responsive and modern dashboard and interfaces using TailwindCSS, enhancing visual appeal and usability across devices.
• Enabled key functionalities including resident and apartment management, grievance submission and tracking, rent payment processing, and notification history for administrators and users.
